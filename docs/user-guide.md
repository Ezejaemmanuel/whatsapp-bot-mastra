## Overview

The WhatsApp Currency Exchange Bot provides a comprehensive admin dashboard for managing customer conversations, transactions, exchange rates, and system settings. The interface is designed to work seamlessly on both desktop and mobile devices.

### Key Features
- **Real-time Chat Management**: Monitor and manage customer conversations
- **Transaction Tracking**: View and update transaction statuses
- **Exchange Rate Management**: Set and update currency exchange rates
- **Bank Details Configuration**: Manage admin bank account information
- **AI Instructions**: Customize the bot's behavior and responses
- **Mobile-Responsive Design**: Full functionality on all devices

## Chat Management

### Viewing Conversations
1. Click on the **Chats** tab in the sidebar
2. Browse the list of conversations
3. Click on any conversation to view details

### Conversation Features
- **Real-time Messages**: See messages as they arrive
- **Message Status**: Delivered, read, and response indicators
- **Customer Information**: View customer details and history
- **Response Management**: Monitor AI responses and intervene when needed

### Chat Interface Elements
- **Message Bubbles**: Customer messages (left), bot responses (right)
- **Timestamps**: When each message was sent/received
- **Status Icons**: Message delivery and read status
- **Media Support**: View images and documents shared by customers

### Managing Conversations
- **Mark as Read**: Conversations automatically mark as read when viewed
- **Search**: Use the search function to find specific conversations
- **Filter**: Sort conversations by status, date, or customer

## Transaction Management

### Viewing Transactions
1. Click on the **Transactions** tab
2. Browse the transaction list
3. Click on any transaction for detailed view

### Transaction Details
Each transaction shows:
- **Transaction ID**: Unique identifier
- **Customer Information**: Name and contact details
- **Currency Pair**: From and to currencies
- **Amount**: Transaction value
- **Exchange Rate**: Applied rate
- **Status**: Current transaction state
- **Timestamps**: Creation and update times

### Transaction Statuses
- **Pending**: Awaiting processing
- **In Progress**: Currently being processed
- **Completed**: Successfully finished
- **Cancelled**: Transaction cancelled
- **Failed**: Transaction failed

### Updating Transaction Status
1. Open the transaction details
2. Click on the status dropdown
3. Select the new status
4. Confirm the update

### Transaction Actions
- **View Details**: See complete transaction information
- **Update Status**: Change transaction state
- **View Related Messages**: See associated chat conversation
- **Download Receipt**: Generate transaction receipt

## Exchange Rates

### Managing Rates
1. Navigate to the **Rates** section
2. View current exchange rates
3. Update rates as needed

### Rate Configuration
- **Currency Pairs**: Set rates for different currency combinations
- **Buy/Sell Rates**: Configure different rates for buying and selling
- **Rate Validity**: Set expiration times for rates
- **Availability**: Enable/disable specific currency pairs

### Updating Rates
1. Click on the rate you want to modify
2. Enter the new rate value
3. Set validity period if needed
4. Save the changes

## Bank Details Management

### Configuring Bank Accounts
1. Go to the **Bank** section
2. Add or edit bank account information
3. Set account preferences

### Bank Account Information
- **Account Name**: Bank account holder name
- **Account Number**: Bank account number
- **Bank Name**: Financial institution name
- **Routing Information**: SWIFT/routing codes
- **Account Type**: Checking, savings, etc.

### Managing Multiple Accounts
- **Add Account**: Create new bank account entries
- **Edit Account**: Modify existing account details
- **Set Default**: Choose primary account for transactions
- **Deactivate**: Temporarily disable accounts

## AI Settings

### Accessing AI Instructions
1. Click on **Settings** in the sidebar
2. Navigate to AI Instructions
3. Edit the system prompt

### AI Configuration Features
- **System Instructions**: Define how the bot should behave
- **Response Templates**: Set standard response patterns
- **Conversation Flow**: Configure interaction logic
- **Language Settings**: Set preferred communication language

### Editing AI Instructions
1. Click in the text area
2. Modify the instructions using Markdown formatting
3. Preview changes if needed
4. Click **Save Changes** to apply

### Best Practices for AI Instructions
- **Be Specific**: Provide clear, detailed instructions
- **Use Examples**: Include sample conversations
- **Set Boundaries**: Define what the bot should and shouldn't do
- **Regular Updates**: Review and update instructions based on performance

## Troubleshooting

### Common Issues

#### Connection Problems
- **Check Internet**: Ensure stable internet connection
- **Refresh Page**: Reload the dashboard
- **Clear Cache**: Clear browser cache and cookies

#### Chat Not Loading
- **Check Status Indicator**: Verify system is online
- **Refresh Conversations**: Pull to refresh or reload
- **Contact Support**: If issues persist

#### Transaction Updates Not Saving
- **Check Permissions**: Ensure you have admin access
- **Verify Data**: Check all required fields are filled
- **Try Again**: Attempt the update again

#### AI Not Responding
- **Check AI Settings**: Verify instructions are configured
- **Review Logs**: Check for error messages
- **Restart Service**: Contact technical support

### Getting Help

#### Support Channels
- **Documentation**: Refer to technical documentation
- **System Logs**: Check application logs for errors
- **Admin Panel**: Use built-in diagnostic tools

#### Reporting Issues
When reporting problems, include:
- **Error Messages**: Any error text displayed
- **Steps to Reproduce**: What you were doing when the issue occurred
- **Browser Information**: Browser type and version
- **Screenshots**: Visual evidence of the problem

### Performance Tips

#### Optimizing Performance
- **Close Unused Tabs**: Reduce browser memory usage
- **Regular Updates**: Keep browser updated
- **Stable Connection**: Use reliable internet connection
- **Clear Storage**: Periodically clear browser storage

#### Best Practices
- **Regular Backups**: Export important data regularly
- **Monitor Usage**: Keep track of system performance
- **Update Settings**: Review and update configurations
- **Train Staff**: Ensure all users understand the interface
